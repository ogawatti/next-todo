# Next TODO Application

This sample app assumes that ALB and Amazon Cognito authentication has been passed.

## Technology used

- Next.js (14.2.x)
- Prisma (5.x)
- MySQL (8.x)

## Installation

```
$ yarn
```

## Setup

```
$ docker-compose up --build
$ yarn db:migrate
```

## Running the app

```
$ docker-compose up --build
$ yarn dev
```
